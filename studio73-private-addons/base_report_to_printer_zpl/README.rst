==========================
Base Report To Printer ZPL
==========================

Módulo para poder imprimir informes de etiquetas en formato ZPL.

- Añade la casilla `Imprimir en ZPL` en informes de tipo `Texto`. Esta casilla debe
  estar marcada para imprimir correctamente etiquetas ZPL. De lo contrario, imprimirá
  el código ZPL como texto en la etiqueta.

- Añade un template de etiqueta ZPL que facilita la creación de etiquetas con logos.
