==============================
Stock Put In Pack Set Qty Done
==============================

Si se activa la opción `Asignar cantidades al poner en paquete` en el tipo de operación
cuando se lance la acción `Poner en paquete` se asignará la cantidad reservada a hecho.
Este es el funcionamiento por defecto de Odoo cuando en el albarán no hay ninguna linea
con cantidad hecha.

Por el contrario, si el check está desmarcado, solo se pondrá en paquete aquellas lineas
que tengan cantidad hecha y en caso de no haber ninguna saltará un aviso indicando que
es preciso añadir las cantidades.
