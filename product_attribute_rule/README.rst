======================
Product Attribute Rule
======================

- Poder crear combinaciones / restricciones basadas en valores de atributo 
  y restringir la creación de variantes de producto

Configuration
=============

- Ir a Ventas - Productos - Configuración - Reglas de atributo para crear nuevas reglas:
   
   - Se puede hacer una regla de tipo "Combinar" y "Restringir", que al final hacen lo mismo, pero permite configurarlo de forma más simple
   - Con una regla de tipo "Combinar", solo se crearán las combinaciones seleccionadas
   - Con una regla de tipo "Restringir", solo se crearán las combinaciones que no estén seleccionadas

- Al crear valores de atributo adicionales no se crearán las combinaciones nuevas automáticamente, es necesario
  pulsar el botón de "Actualizar lineas" en la cabecera, que generará las combinaciones
