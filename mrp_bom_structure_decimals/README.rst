==========================
MRP BoM Structure Decimals
==========================

- Añade una precisión decimal "Lista Materiales Costes" para ajustar la cantidad de decimales en la vista de Estructura de Lista de Materials & Costo
- También cambian en el informe PDF
- Cambia el PDF a estar en vertical
