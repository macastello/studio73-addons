===================================
Stock Barcode Location Product Info
===================================

Módulo que añade un menú nuevo que permite poder escanear un producto o una ubicación.
  - Si se escanea un producto, se muestra todo el stock que tiene dicho producto en 
    una vista de ubicaciones simple (ubicación, cantidad disponible)
  - Si se escanea una ubicación, se muestra todo el stock que hay en dicha ubicación 
    agrupado por producto.
  - Se establece la vista predeterminada a tipo kanban.
