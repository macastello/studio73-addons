========================
Account Payment Advanced
========================

Permite la creación de pagos mediante una interfaz mejorada por nosotros, dando la
posibilidad de pagar varias facturas a la vez asginando a cada una el importe pagado.

Configuration
=============

**No calcular facturas**: Para casos en que hay muchas facturas a pagar y no se desea
que se intente repartir la cantidad a pagar entre ellas.
**Agrupar pagos**: Si se quiere agrupar en un único pago las facturas del mismo cliente
**Permitir conciliar apuntes**: Para que nos aparezcan también para pagar asientos de
tipo 'entry'.
