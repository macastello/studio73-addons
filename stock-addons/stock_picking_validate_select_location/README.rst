======================================
Stock Picking Validate Select Location
======================================

- El módulo permite seleccionar una ubicación de destino para el albarán actual y sus albaranes
  'hermanos' al validar si aún no se ha seleccionado ninguna.


Configuration
=============

- Para que se muestre la ventana emergente de selección de ubicación se debe activar el campo en
  tipos de operaciones `Elegir ubicación al validar agrupación`.

      - **Elegir siempre**: Se ha de establecer la ubicación en cada validación para las 
      las líneas que se están validando

      - **Elegir una vez**: Se ha de establecer la ubicación en la primera validación de
      la agrupación. El valor se traspasará al resto de albaranes de la agrupación pendientes.
